URL based math operations!

History :

  • You can view upto 20 recent operations in the /history command

    •         
        URL : http://localhost:3000/history

    Mathematical Operations such as :

  • Addition

    •         
        Syntax : http://localhost:3000/operand1/plus/operand2

        Example :http://localhost:3000/5/plus/3

  • Subtraction

    •         
        Syntax : http://localhost:3000/operand1/minus/operand2

        Example : http://localhost:3000/5/minus/3

  • Multiplication

    •         
        Syntax : http://localhost:3000/operand1/into/operand2

        Example : http://localhost:3000/5/into/3
